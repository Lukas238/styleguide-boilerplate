5.1 #Mixings - Breakpoint Breakpoint

Wrap code in a media query block for any of the available media dimensions.

Media dimensions:

  • $xs: Mobile and up
  • $sm: Tablet and up
  • $md: Dektop and up
  • $lg: Large Desktop
  • $xs-sm: Mobile Only
  • $sm-only: Tablet only
  • $md-only: Desktop only
Parameters:
  • $map
    The media dimension variable to use, or a sass map with a max and minkeys. Ex.: (min: 480px, max: 512px).
